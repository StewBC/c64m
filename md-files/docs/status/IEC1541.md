# IEC / 1541 status

## Current implementation

- VIA 6522 module implemented and unit-tested (Phase 1 complete).
- 1541 machine shell implemented (Phase 2 complete):
  - `c1541` struct: 6502 CPU (C6510 reuse), VIA #1, VIA #2, RAM (2 KB + mirror), ROM (16 KB)
  - Bus read/write with correct VIA #2-before-VIA-#1 priority
  - IRQ via VIA #1 and VIA #2 IFR; NMI via VIA #2 CA1 (ATN line)
  - ROM loading: `c1541_load_rom()` (single 16 KB file) and `c1541_load_rom_split()` (two 8 KB halves)
  - ROM discovery follows existing search-path pipeline (`"."`, `"rom"`, `"roms"`); absent ROM is non-fatal
  - Drive 8 and Drive 9 integrated into `c64_t`; stepped once per C64 clock cycle
  - `advance_one_cycle()` is a no-op when ROM is not loaded

## Known limitations / deferred

- IEC bus wiring between C64 CIA #2 and 1541 VIA #2: Phase 3.
- NTSC clock divider: Phase 3 (PAL 1:1 lockstep only for now).
- Drive mechanics (VIA #1 head stepper, motor, SYNC): Phase 3+.
